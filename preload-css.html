preload-css

이 페이지는 필수 CSS가 임베디드 형식으로 <style>...</style> 삽입되어 있습니다. 사용하지 않는 CSS 코드는 병렬(preload) 로딩 후 지연(onload) 적용합니다. Performance 탭을 확인하세요.

<link rel="preload" as="style" href="unused-style-only.css" onload="this.onload=null;this.rel='stylesheet'">
